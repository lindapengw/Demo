1.System requirement
a. Windows operation system
b. Browser: Firefox version under 91 is preferable (test in version 87); Chrome Version 91.0.4472.101
c. Apache Maven 
d. Install TestNg plugin in Eclipse. https://testng.org/testng-eclipse-update-site

2. Execute test cases with Maven via command line
a. Download "Demo" project to local drive
b. Execute Maven command: "C:\Demo>mvn clean install"

3. Test Result 
a. Report directory: "C:\Demo\target\reports\report.html"
b. Screenshot of failed test cases is located: "C:\Demo\target\reports\screenshot"
